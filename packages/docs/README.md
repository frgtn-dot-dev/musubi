# Musubi Starlight documentation

This package is the canonical source for <https://musubi.pro/docs/>.

```sh
pnpm docs:dev
pnpm docs:build
```

Content lives in `src/content/docs/`; the explicit sidebar and deployment base
path live in `astro.config.mjs`. Read the
[documentation contribution guide](./src/content/docs/contributing/documentation.mdx)
before adding or reorganizing a page.
