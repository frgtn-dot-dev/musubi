Musubi is a collaborative calendar app for mobile. The name comes from the Japanese word 結び (*musubi*) meaning "knot" or "bond" — the app ties people's schedules together through shared calendars.

## Key concepts

**Calendars** are the main organizational unit. A user can own multiple calendars and invite others to join them. Events live inside calendars.

**Events** belong to a single calendar. They have a title, start/end time, optional description, and are tied to all members of that calendar.

**Invites** are how users join a calendar. An owner sends an invite link or email; the recipient accepts it inside the app.

**Users** authenticate via email/password. Each user has a profile with a display name.

## Tech stack

| Layer | Technology |
|---|---|
| Mobile client | React Native + Expo |
| Routing | Expo Router (file-based) |
| State | Zustand |
| Server | Node.js REST API |
| Auth | Better Auth |
